## Local Development

### Overview

This repository hosts the PagerDuty mApp file for the Cherwell ITSM integration. The `.mApp` file is a Cherwell Trebuchet Blueprint XML artifact that is imported directly into a Cherwell instance.

### Prerequisites

- Access to a Cherwell instance with admin permissions
- The full integration guide: https://www.pagerduty.com/docs/guides/cherwell-integration-guide/

### Repository Contents

| File | Description |
|------|-------------|
| `PagerDuty.mApp` | Cherwell mergeable application package (Trebuchet Blueprint XML) |
| `README.md` | Repository overview and link to integration guide |

### Working with the mApp File

The `PagerDuty.mApp` file is a large XML file generated by the Cherwell Administration Console. It contains image definitions, form layouts, one-step actions, and other Cherwell configuration objects for the PagerDuty integration.

To install or update the integration:
1. Download `PagerDuty.mApp` from this repository
2. Open the Cherwell Administration Console
3. Import the mApp using the Cherwell mApp import wizard
4. Follow the integration guide for configuration steps
